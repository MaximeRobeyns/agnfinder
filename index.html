Welcome to the agnfinder documentation!

For instructions on how to install the project, please see the installation page. This page contains information for both users and developers.

For configuration guidance and general usage, please see the configuration page.

This is a fork of the original AGNFinder project by Mike Walmsey and collaborators, which aims to use (conditional) generative modelling techniques to map from photometry to distributions over physical parameters \(\theta\).

There are broadly two components;

  • The first is the sampling code, where a dataset of (\(\theta\), photometry) pairs are created, using Prospector. More information about the sampling process is in the sampling page.

  • The second focuses on the task of inferring galaxy parameters from photometry, using the generated datasets from the previous sampling component. For a description of the methods used and a guide to writing new models or adapting some of the existing ones, please see the inference page.
